# M5Stack Cardputer Companion Satellite Controller

Professional firmware to transform your M5Stack Cardputer into a wireless control surface for Bitfocus Companion.

## Features

- 🎮 **Full keyboard navigation** - Navigate all interface with physical keyboard
- 📱 **Professional UI** - Stream Deck inspired interface with animations
- 🔌 **Real-time Companion integration** - WebSocket communication with Bitfocus Companion
- ⚙️ **Easy setup** - On-device configuration wizard
- 🔄 **Auto-reconnection** - Automatic WiFi and Companion reconnection
- 📄 **Multi-page support** - Up to 10 pages of buttons
- 🎨 **Visual feedback** - Button highlights and press animations
- 💾 **Persistent settings** - Saves configuration in NVS memory
- 🚀 **RTOS based** - Multi-tasking for smooth performance

## Hardware Requirements

- M5Stack Cardputer v1.1 (ESP32-S3)
- USB-C cable for programming

## Software Requirements

- PlatformIO IDE or CLI
- Bitfocus Companion (latest version) running on your computer
